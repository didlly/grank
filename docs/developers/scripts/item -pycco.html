The has_item function checks if the account has an item. It takes in a string as an argument, which is the name of the item to be checked for. The function then sends a message asking for information about the item, and parses the number of owned items from the embed.

Args: Client (Instance): The Discord client item (str) The item to be checked

Returns: bool: Indicates whether the account has the item or not
