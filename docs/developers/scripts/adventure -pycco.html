The adventure function is used to interact with the adventure command. It will start a new adventure if one is not already in progress, or it will continue an existing one. If the cooldown period hasn’t ended, it will return False

Args: Client (Instance): The Discord client

Returns: bool: Indicates whether the command ran successfully or not
